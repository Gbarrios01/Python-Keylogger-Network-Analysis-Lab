## Overview
This project demonstrates a **controlled malware simulation** in a virtual lab environment.  
Keystrokes typed on a **victim VM** are captured and sent to an **attacker VM** using Python. The data exfiltration is monitored and analyzed using **Wireshark**.

This lab is fully isolated and **designed for educational cybersecurity research**.

## Attacker Setup

1. Open a terminal on the **Attacker VM**.  
2. Navigate to the attacker folder and start the listener:

`cd malware_lab`  
`python3 listener.py`

## Victim Setup

1. On the **Victim VM**, open terminal.  
2. Navigate to the victim folder and start the keylogger:

`cd keylogger_lab`  
`sudo python3 bad_script.py`

## Network Traffic Analysis

Network traffic generated by the keylogger can be analyzed using **Wireshark**.

1. Open Wireshark on the attacker VM:

`sudo wireshark`

2. Select the host-only network interface.  
3. Apply filter:

`tcp port 4444`

4. Start capturing packets and type on the victim VM.  
5. Use **Follow → TCP Stream** to observe transmitted keystrokes.
